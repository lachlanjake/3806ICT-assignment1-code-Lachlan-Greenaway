# 3806ICT-assignment1-code-Lachlan-Greenaway
This repository contains the source code, generated datasets, and experiment results for Assignment 1.

## Files

- `src/main.py` contains the parser, simplifier, baseline Algorithm 2 implementation, improved method, and experiment runner.
- `datasets/` contains the generated formula datasets.
- `results/results.csv` contains the experiment results.

## How to run

Run the experiment with:

python3 src/main.py

The program reads the dataset files and saves the results to `results/results.csv`.
